# Sales-Dashboard

# Overview
Sales Dashboard visualizing FY2025 sales data across zones and product categories.

# Features
- Live KPI strip: Total Revenue, Units Sold, Net Gain, Transactions, and Margin Rate, each with animated count up and month over month delta indicators (▲/▼).
- Product filter: dropdown to switch between All Products, Furniture, Electronics, and Stationery & Office, with every KPI/chart recalculating instantly.
- 4 custom canvas gauges: circular performance score, donut chart (revenue by zone), horizontal lead-quality index, and semi-circle target-achievement gauge.
- 3 custom canvas charts: stacked bar (revenue by zone/segment), smoothed area/line chart (monthly revenue trend), and bar chart (monthly transactions).
- Interactive tooltips: hover states on all charts showing contextual values.
- Metric pills: Win Rate, Avg Deal Size, Pipeline Value as derived/estimated metrics.
- Fully responsive canvas redraw on window resize.
- Dark, glassmorphism-style UI with a custom color system via CSS variables.
